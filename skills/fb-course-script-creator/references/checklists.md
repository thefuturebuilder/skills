# Admin Checklists

Include these at the end of every completed course script document.

## Co-Creator Course Checklist

- [ ] Course title based on the course outcome
- [ ] Lesson names based on the to-dos to achieve the outcome
- [ ] Write lesson scripts before recording in Tasmania
- [ ] Write the ad, YouTube explainer and course intro scripts before recording in Tasmania
- [ ] Record lesson and other course related videos in the Tasmania studio

## Course and Lesson Checklist

- [ ] Course Categories (Subcategory only if needed)
- [ ] Lesson Name (per lesson requirement)
- [ ] Post Video
- [ ] Post Title
- [ ] Associated Category
- [ ] Post Body Text
- [ ] Post Thumbnail (use .jpg or .png with non-transparent background — dimensions of 1280x720)
- [ ] Post Files
- [ ] Assessment
- [ ] Extra Video Content Per Lesson
  - [ ] FAQs — answer questions that come up all the time, once and for all
  - [ ] Principles and Key Insights
  - [ ] Anecdotes/Story (journey: situation before → call to adventure → journey struggle → victory and new normal)
  - [ ] Deep Dive/Geek Out/Guide on something you are fascinated about
  - [ ] Application and Adaptation
  - [ ] Case study (proof, case studies, timelines, data & stats)
  - [ ] Live Performance Commentary
  - [ ] Trend (Past, present, future)
  - [ ] Disruptive Idea
  - [ ] Common Objections

## Course Details Checklist

- [ ] Course Title (140 characters max, outcome-based, ideally rhymes)
- [ ] Course Description
- [ ] Product Thumbnail (Recommended dimensions of 1280x720)
- [ ] Logo Image (Recommended dimensions of 640x640)
- [ ] Favicon Image (Recommended dimensions of 32x32)
- [ ] Any Custom JS or Custom CSS
- [ ] Tracking Code for Header and Footer
- [ ] Certificates
- [ ] Certificates Title
- [ ] Associated Course
- [ ] Assign the course category
- [ ] Assign all relevant tags for the content
